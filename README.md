
# Steemit üzerindeki SBD ve STEEM'lerin TL olarak toplam deðeri

Sahip olduðunuz Steem ve SBD'lerin deðerini TL olarak görebilirsiniz.

#### Kurulum

* Pyhton 2.7 indirin https://www.python.org/downloads/
Aþaðýdaki þekilde kütüphaneleri yükleyin
* pip install os
* pip install requests
* pip install re



#### Kullaným
* **SbdSteemTL.py** dosyasýný python IDE ile açýn
* F5'e basýn
* Steemit kullanýcý adýnýz girin
